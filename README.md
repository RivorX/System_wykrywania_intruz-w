# System wykrywania intruzów

Krótki projekt do budowy i testowania mechanizmów wykrywania nieautoryzowanej aktywności na podstawie danych wejściowych.

## Cel
- przygotowanie pipeline'u danych,
- trenowanie i ewaluacja modelu wykrywania intruzów,
- uruchamianie testów na nowych próbkach.

## Struktura projektu
- `config/` - konfiguracje eksperymentów i parametrów,
- `data/` - dane wejściowe i zbiory testowe,
- `scripts/` - skrypty do trenowania, ewaluacji i inferencji.

## Wymagania
- Python 3.10+,
- środowisko wirtualne (`venv`),
- zależności projektu (`requirements.txt`).

## Status
Projekt jest w fazie przygotowania - struktura repo jest gotowa, a skrypty i konfiguracje będą rozwijane iteracyjnie.
